# Cury Light Language

Animation scripts for Cury UI and ambient lighting.

## Files

- `cury-light-language-2.shelly.js`
  - Use case: State-driven communication model with explicit modes (boot/ready/touch/pairing/active/low_liquid/error/night/goodbye).
  - User persona: Product firmware engineer aligning LEDs to device lifecycle/state machine.
- `cury-light-language-v2.shelly.js`
  - Use case: Continuous visual loop with color cycling and crossfade dynamics for ambient presentation.
  - User persona: Experience designer creating expressive demo or showroom light behavior.

## Relationship

These two scripts are variants, not duplicates.
Use only one animation script in the target script slot at a time.
